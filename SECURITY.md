# Security Policy

Bu repository egitsel bir laboratuvar icerir. Ucuncu taraf sistemlerde test yapmayin, Appwrite Cloud veya kendi self-hosted ortaminiz disinda hedef secmeyin.

## Supported Scope

- `web/` icindeki demo arayuzu
- `functions/` icindeki laboratuvar Function kodlari
- `scripts/` icindeki kurulum ve temizlik otomasyonlari
- `docs/` icindeki analiz ve savunma dokumani

## Out of Scope

- `appwrite/appwrite` upstream reposunda dogrulanmamis acik iddiasi
- Ucuncu taraf Appwrite instance'lari
- Yetkisiz gercek veri denemeleri

## Disclosure

Upstream Appwrite cekirdegi ile ilgili gercek bir guvenlik bulgusu tespit ederseniz resmi sureci izleyin:

- [Appwrite Security Policy](https://github.com/appwrite/appwrite/security/policy)
